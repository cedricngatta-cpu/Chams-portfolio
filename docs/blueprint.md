# **App Name**: No-Code AI Portfolio

## Core Features:

- Hero Section: Display a futuristic hero section with a catchy title, subtitle, and neon call-to-action button linking to the projects.
- About Section: Showcase a brief professional background, passion for AI and No-Code, and approach to development.
- Skills & Tools: List key skills and tools ('Lovable', 'Stitch', 'Make', 'Bubble', 'OpenAI', 'Firebase') with futuristic icons.
- AI Project Showcase: Display a grid of projects. Each card includes the project title, description, used tools (e.g., Lovable, Stitch, OpenAI), and a link to project details. It also contains a thumbnail of the project, dynamically generated with AI tool when the user creates a new project.
- Contact Section: Include a minimal contact form (Name, Email, Message) and links to LinkedIn and GitHub.

## Style Guidelines:

- Background: Deep dark mode using shades of midnight blue (#191970), anthracite gray (#36454F) and black (#000000).
- Glassmorphism: Apply a pronounced glassmorphism effect to cards, sections, and interactive elements.
- Accents: Vibrant neon colors (electric blue (#7DF9FF), intense violet (#9400D3), lime green (#32CD32)) for call-to-action buttons, icons, and important titles.
- Font: 'Inter' (sans-serif) for the main text and 'Source Code Pro' (monospace) for code snippets.
- Use futuristic and minimalist icons.
- Implement smooth micro-interactions on hover, subtle transitions between sections, and light particle effects in the background.
- Apply generous spacing for optimal readability.
