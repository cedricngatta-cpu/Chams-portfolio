# 🚀 Chams Portfolio | AI Architect

Ce projet est un portfolio futuriste pour un développeur AI No-Code, conçu avec **Next.js**, **Tailwind CSS**, **ShadCN UI** et **Genkit** pour les fonctionnalités d'IA.

## ✨ Caractéristiques
- **Design Anti-Gravity** : Interface épurée avec effets de parallaxe et de flottement.
- **Génération d'Images IA** : Utilisation de Genkit et Imagen 4 pour générer des miniatures de projet à la volée.
- **Composants Glassmorphism** : Esthétique moderne avec flous d'arrière-plan et bordures néon.

## 🛠️ Installation

1. **Cloner le projet**
   ```bash
   git clone https://github.com/cedricngatta-cpu/Chams-portfolio.git
   cd Chams-portfolio
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

## 🛰️ Déploiement sur GitHub (cedricngatta-cpu)

Si vous avez l'erreur `Repository not found`, suivez exactement ces étapes :

1. **Création sur GitHub** : Allez sur [github.com/new](https://github.com/new) et créez un dépôt nommé exactement `Chams-portfolio`. Ne cochez **aucune** case (pas de README, pas de licence).
2. **Lier le code local** : Ouvrez votre terminal dans le dossier du projet et lancez :
   ```bash
   # Si vous n'avez pas encore initialisé git
   git init
   
   # Ajouter les fichiers
   git add .
   git commit -m "Initial commit: Chams Portfolio"
   
   # Configurer la branche principale
   git branch -M main
   
   # Lier à votre dépôt (Vérifiez bien l'URL)
   git remote remove origin 2>/dev/null
   git remote add origin https://github.com/cedricngatta-cpu/Chams-portfolio.git
   
   # Envoyer le code
   git push -u origin main
   ```
